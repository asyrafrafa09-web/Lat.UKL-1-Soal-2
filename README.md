# Program untuk menentuakan apakah bilangan tersebut ganjil atau genap
Pengguna diminta untuk memasukkan sebuah bilangan, lalu program memeriksa dengan rumus bilangan % 2.
Jika hasilnya 0, maka bilangan genap. jika tidak, bilangan ganjil.
Hasilnya kemudian ditampilkan di layar.
